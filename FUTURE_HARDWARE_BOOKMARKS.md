# Future Hardware Bookmarks

Keep these for the first real bench test:

- Confirm the ESP32 serial path on the Pi (`/dev/ttyACM0`, `/dev/ttyUSB0`, or `/dev/serial/by-id/...`).
- Verify the Inland relay module switches reliably from the chosen logic level and power rail.
- Add a proper valve test load before controlling a live irrigation valve.
- Replace the synthetic camera with the real camera feed and validate lighting at sunrise / sunset.
- Install and calibrate the real soil moisture sensor.
- Install and calibrate the real pressure sensor / pressure switch.
- Re-enable autonomy only after the above safety checks pass.
